# This directory contains media assets for the portfolio website.
# 
# To add media files:
# - Place image files (e.g., .jpg, .png, .gif) in the `images` folder.
# - Place video files (e.g., .webm, .mp4) in the `videos` folder.
# 
# Guidelines:
# - Ensure that image files are optimized for web use to improve loading times.
# - Use appropriate video formats (.webm or .mp4) for compatibility across different browsers.
# - Maintain a consistent naming convention for easy identification of files.
