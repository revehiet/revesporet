# Portfolio Website

## Overview
This is a basic portfolio website project that showcases media files such as images and videos in a collage layout. The website is designed to be simple and easy to customize.

## Project Structure
```
portfolio-website
├── src
│   ├── index.html
│   ├── styles
│   │   └── style.css
│   └── assets
│       ├── images
│       ├── videos
│       └── README.md
├── README.md
└── .gitignore
```

## Features
- Collage layout for displaying images and videos.
- Hover effects that reveal text descriptions for each media item.
- Responsive design that adapts to different screen sizes.

## Setup Instructions
1. Clone the repository to your local machine.
2. Navigate to the `portfolio-website` directory.
3. Open `src/index.html` in a web browser to view the portfolio.
4. Add your media files to the `src/assets/images` and `src/assets/videos` directories.

## Customization
- To change the layout or styles, modify the `src/styles/style.css` file.
- To add new media items, place your image files in the `src/assets/images` folder and video files in the `src/assets/videos` folder.
- Update the `src/index.html` file to include any new media items you want to showcase.

## Guidelines for Media Files
- Images should be in formats such as .jpg, .png, or .gif.
- Videos should be in .webm or .mp4 format for compatibility.
